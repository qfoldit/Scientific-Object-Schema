# Scientific World Schema v1.0

Scientific World Schema is the foundation layer for a persistent
scientific multiplayer world.

Unlike a simple data format, it models scientific activity.

The system describes:

1.  What exists
2.  What can be done
3.  Who performs actions
4.  How experiments evolve
5.  How knowledge is produced

## Parametric Biology

Biological systems are dynamic.

They can grow, mutate, evolve and respond to environments.

The schema supports L-systems, genetic parameters, environmental
feedback and procedural biological development.

## Applications

-   Foldit-style protein design
-   drug discovery
-   synthetic biology
-   ecological simulations
-   materials discovery
-   AI scientist agents
